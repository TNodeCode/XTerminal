# XTerminal

Lightweight utilities to run local and remote (SSH) processes with event hooks, output capture and simple terminal wrappers.

## Installation

```bash
pip install xterminal
```

PyPi: https://pypi.org/project/xterminal/

## Overview

This project exposes a small set of classes to run processes and SSH commands programmatically while providing event hooks for stdout/stderr lines and execution lifecycle events.

See the Usage page for quick examples and the API section for the full reference.
